Privacy Policy

Information we collect

  • User Name: Users are advised to provide their nick name and avoid real names as far as possible. This name is used on the dashboads and leaderboards to indicate the User's position
  • User Email: Users provide their email as part of the registration so that the LoF application can send emails to complete the registration, reset passwords in the future and also to use the email as an unique identifier for login
  • User Password: Password to login to the LoF application. This is only known to the User and can be reset by the user with the reset link sent to the User provided email address
  • User Activity Prefferences: Data entered by the user for various activity categories like Exercise, Lifestyle, Diet, etc. as agreed by the User's league. Some data will be pulled in from fitness apps like Strava based on the User's consent.

    •  

    How we use the collected information

  • Goal Tracking: The activities entered by the User are logged and used to track the progress against goals for each activity set by the user on a weekly basis. This progress is only visible to the User
  • Leaderboards: The maximum aggregated activity score and maximum individual activity record are presented on a Leaderboard with the User's name. This will only be visible to Users of the specific league
  • Teams: If the Users of a league are split into Teams, then the aggegated scores of the members in the teams are also presented on the Teams leaderboard

    •

  • Super Admin will have access to data logged by Users and can appply manual corrections as needed on a case by case basis
